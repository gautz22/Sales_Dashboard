📌 Project Overview

This project is an interactive Sales Dashboard built using Microsoft Excel.
The dashboard provides insights into sales performance, profit trends, customer distribution, and monthly growth patterns.

It helps businesses understand:

Sales performance by category

Yearly profit trends

Sales distribution across states

Customer growth over time

Top performing customers

Monthly sales trends

🎯 Objectives

Transform raw sales data into meaningful insights

Create interactive filters using slicers

Visualize trends using charts and maps

Design a clean and professional dashboard layout

🛠️ Tools & Techniques Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers (Category & Year filters)

Conditional Formatting

Map Visualization

Data Aggregation & Cleaning

📊 Dashboard Features
1️⃣ Sales by Category

Displays total sales across product categories

Helps identify high-performing product segments

2️⃣ Profits by Year

Shows yearly profit trends

Compares Furniture, Office Supplies, and Technology categories

3️⃣ Sales by State

Geographic visualization of sales distribution across U.S. states

4️⃣ Customer Count per Year

Tracks customer growth trend over time

5️⃣ Top 5 Customers

Identifies highest revenue-generating customers

6️⃣ Sales by Month

Shows monthly sales pattern

Helps identify seasonal trends

📈 Key Insights

Technology category shows strong profit growth over the years.

Customer count is increasing steadily.

Certain states contribute significantly more to total revenue.

Monthly sales show peak periods indicating seasonal demand.
